> Rewrite the **WHERE** clause
```sql 
WHERE UNIQUE (SELECT title FROM course)
```
> without using the **UNIQUE** construct. 

--------------------------------

One method

```sql
WHERE 1 >= ALL (
    SELECT COUNT(*)
    FROM course
    GROUP BY title
)
```

Another method

```sql 
WHERE NOT EXISTS (
    SELECT * 
    FROM course AS c1, course AS c2
    WHERE c1.course_id != c2.course_id AND c1.title = c2.title
)
```
